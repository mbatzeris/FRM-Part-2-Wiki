## How to Update This Radar

1. **After each study session:** Update LO status in `04_LO_TRACKER.md`
2. **After each practice question:** Log errors in `07_ERROR_LOG.md`
3. **Weekly:** Run this formula to re-rank:
   - Status score: `[ ]`=0, `[~]`=0.33, `[✓]`=0.67, `[★]`=1.0
   - Weakness = (1 - status_score) × domain_weight × (1 + error_count)
4. **Sort:** Highest weakness score = top priority
